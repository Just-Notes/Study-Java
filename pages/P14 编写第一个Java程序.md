- 编写一个 HelloWorld 程序
	- public class HelloWorld {
	      public static void main(String[] args) {
	          System.out.println("Hello, World!");
	      }
	  }
-
- 编译 Java 程序
	- ```bash
	  ❯ javac HelloWorld.java 
	  ```
-
- 运行第一个 Java 程序
	- ```bash
	  ❯ java HelloWorld 
	  Hello, World!
	  ```
