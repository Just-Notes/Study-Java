- #Java #class #注释
- 注释有什么用？
	- • Java中的注释是用于解释和说明代码的文本，，它不会被编译器编译，也不会被程序执行。
	- • 注释可以帮助程序员快速解读程序。
-
- 单行注释
	- ```java
	  // 这是一个单行注释
	  ```
-
- 多行注释
	- ```java
	  /*
	  多行注释
	  多行注释
	  多行注释
	  多行注释
	  多行注释
	  */
	  ```
-
- javadoc
	- ```java
	  /**
	   * HelloWorld 
	   * @author Aaron
	   * @version 1.0
	   * @since 2024-05-03
	   */
	  
	  public class HelloWorld {
	  
	      /**
	       * 这是一个main方法，是程序的入口
	       * @param args 这个是main方法的参数
	       */
	      public static void main(String[] args) {
	          System.out.println("Hello, World!");
	      }
	  }
	  ```
